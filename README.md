# 🎓 Gamified AI Study Companion

An AI-powered learning assistant that helps students understand concepts, generate quizzes, and stay engaged through gamified learning.

This prototype was built for the **AI Engineering Internship Round 1** submission for S3K Technologies.

---

## 🚀 Features

* **AI Concept Explanation** – Ask questions and get simple explanations from an AI tutor.
* **Quiz Generation** – Automatically generates practice questions for any topic.
* **Gamification** – Students earn points for answering questions correctly.
* **Interactive Interface** – Simple chat-style interface built using Streamlit.

---

## 🛠️ Tech Stack

* Python
* Streamlit
* OpenRouter API (GPT-3.5 Turbo)
* python-dotenv

---

## ⚙️ Setup

Install dependencies:

```
pip install -r requirements.txt
```

Create a `.env` file:

```
OPENROUTER_API_KEY=your_api_key_here
```

Run the app:

```
streamlit run main.py
```
